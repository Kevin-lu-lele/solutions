1. Resize
I am an example Box "#ex1"
You can *RESIZE* Me.
Javascript 
$().ready(function() {
  $('#ex1').jqResize('.jqResize');
});
CSS 
.jqHandle {
   background: red;
   height:15px;
}

.jqDrag {
  width: 100%;
  cursor: move;
}

.jqResize {
   width: 15px;
   position: absolute;
   bottom: 0;
   right: 0;
   cursor: se-resize;
}

2. Drag
I am an example Box "#ex2"
You can *DRAG* Me.
I am an example Box "#ex2"
You can *DRAG* Me.
Notice my Original Opacity is preserved.
Javascript 
$().ready(function() {
  $('#ex2').jqDrag();
  
  $('#ex2b')
    .css('opacity',0.6)
    .jqDrag();
});
